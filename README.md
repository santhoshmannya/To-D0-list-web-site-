# To-Do list

## Overview

Task Dashboard is a modern and responsive task management web application built using **HTML, CSS, and JavaScript**. It helps users organize daily activities, track progress, and manage tasks efficiently through a clean and user-friendly interface.

## Features

* Add, edit, and delete tasks
* Mark tasks as completed
* Assign categories to tasks
* Set task priorities (High, Medium, Low)
* Add due dates
* Search tasks instantly
* Filter tasks by status (All, Active, Completed)
* Real-time task statistics
* Progress tracking with a dynamic progress bar
* Local Storage support for data persistence
* Responsive design for desktop and mobile devices
* Modern dark-themed dashboard interface

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in any modern web browser.

No additional installations or dependencies are required.
